CustomModels IBM.WatsonDeveloperCloud.LanguageTranslator.v2.ILanguageTranslatorService.CreateModel ( CreateModelOptions  _options)

Uploads a TMX glossary file on top of a domain to customize a translation model. Depending on the size of the file, training can range from minutes for a glossary to several hours for a large parallel corpus. Glossary files must be less than 10 MB. The cumulative file size of all uploaded glossary and corpus files is limited to 250 MB.

Parameters
_optionsAn object containing arguments used to create the custom model.
Returns

TranslateResponse IBM.WatsonDeveloperCloud.LanguageTranslator.v2.ILanguageTranslatorService.Translate ( string  _source,
string  _target,
string  _text 
)

Translates input text from the source language to the target language.

Parameters
_sourceUsed in combination with target as an alternative way to select the model for translation. When target and source are set, and model_id is not set, the system choose a default model with the right language pair to translate (usually the model based on the news domain).
_targetTranslation target language in 2 or 5 letter language code. Should use 2 letter codes except for when clarifying between multiple supported languages. When model_id is used directly, it will override the source-target language combination. Also, when a 2 letter language code is used, and no suitable default is found (such as “zh”), it returns an error.
_textInput text in UTF-8 encoding. Multiple text query parameters indicate multiple input paragraphs, and a single string is valid input.
Returns
