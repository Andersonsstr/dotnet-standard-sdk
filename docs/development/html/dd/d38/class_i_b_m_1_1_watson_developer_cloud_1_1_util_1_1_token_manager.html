string IBM.WatsonDeveloperCloud.Util.TokenManager.GetToken ( )
inline

This function sends an access token back through a callback. The source of the token is determined by the following logic:

  1. If user provides their own managed access token, assume it is valid and send it
  2. If this class is managing tokens and does not yet have one, make a request for one
  3. If this class is managing tokens and the token has expired, refresh it
  4. If this class is managing tokens and has a valid token stored, send it
Returns
An IamTokenData object containing the IAM token.

void IBM.WatsonDeveloperCloud.Util.TokenManager.SetAccessToken ( string  iamAccessToken)
inline

Set a self-managed IAM access token. The access token should be valid and not yet expired.

By using this method, you accept responsibility for managing the access token yourself. You must set a new access token before this one expires. Failing to do so will result in authentication errors after this token expires.
