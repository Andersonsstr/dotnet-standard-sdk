◆ Count

int IBM.WatsonDeveloperCloud.SpeechToText.v1.Model.WordData.Count
getset

Gets or sets a sum of the number of times the word is found across all corpora. For example, if the word occurs five times in one corpus and seven times in another, its count is 12. If you add a custom word to a model before it is added by any corpora, the count begins at 1; if the word is added from a corpus first and later modified, the count reflects only the number of times it is found in corpora.
